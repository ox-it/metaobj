Exporting and importing forms

If you own more than one site, you may want to use a particular form in two or more sites. To do so, you can either recreate the form on the second site from the .xsd file in Resources, or export the form to your desktop as a compressed file and then import it to the other site(s) where you plan to use it. The compressed .zip file contains the XML file defining the editable data fields, Name and Instructions, and the .xsd file comprising the schema. You may also wish to acquire exported .zip files for forms from other users for import to your site(s).

Exporting a form

To export a zipped file for a form, in the Forms tool:

  1. If the Manage Forms page is not displayed, click the reset button ( 
     ).

  2. Below the name of the form you want to export, click Export.

  3. Your browser will prompt you to open or save the file. Save the file to a local storage device. The exported form will have a .zip extension.

Importing a form

To import a form:

  1. Access the site to which you want to import the form.

  2. In the site where you are maintaining forms, upload the compressed .zip file to Resources.

  3. Access the Forms tool. If the Manage Forms page is not displayed, click the reset button ( 
     ).

  4. At the top of the page, click Import.

  5. Click Select File.

  6. Locate the .xsd file you uploaded in step 1. Next to the name of the file, click Select.

  7. Scroll to the bottom of the page and click Continue.

  8. Click Import. The imported form will be listed under its original name with a site state of "unpublished".
