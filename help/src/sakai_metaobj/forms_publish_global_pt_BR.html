Publicar Globalmente um Formulário

Se você é um administrador de sistema, você pode publicar um formulário globalmente (isto é, para todos os sites). Para isso, faça o login e siga estes passos:

  1. Clique na pasta Administração no topo da tela.
  2. Selecione o site PortfolioAdmin.
  3. Acesse a ferramenta Formulários. O Sakai exibirá a página inicial de Formulários.
  4. Encontre o formulário que você deseja publicar e clique no link de aprovar publicação global. O Sakai exibirá essa tela de confirmação:
  5. Clique em Sim. O Sakai tornará o formulário disponível para todos os sites.
