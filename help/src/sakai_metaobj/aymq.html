Editing a form

Using the Forms tool, you can change the name of a form and the instructions for using it. However, you cannot use the tool to change the structure (fields and data types) of the form itself.

To edit the name or instructions for a form:

  1. If you do not see the Manage Forms screen, click the reset button ( 
     ).

  2. Find the form you want to change, and click Edit.

  3. Change the content in the "Name" and "Instruction" boxes as needed. Use the icons and drop-down menus to format the instructions text as desired. To preview the form while editing, click Preview.

  4. When you are finished, click Save Changes to save and return to the Manage Forms page.
