Acessar uma Lista de Formulários

A ferramenta Formulários geralmente está localizada na guia categoria em um site portfolio. Para acessar a ferramenta, clique no botão Iniciar para formulários. O Sakai exibirá a página inicial de Formulários.

A página inicial contém uma lista de formulários que foram criados localmente para o seu site. Os formulários são ordenados alfabeticamente. Para cada formulário, você pode ver o site no qual o formulário é publicado, quando foi modificado pela última vez, e seu estado (publicado, não publicado, ou aguardando aprovação).

Se a lista de formulários for maior do que uma página, use os botões na barra de menu de Formulários para exibir outras páginas na lista.

  • Para exibir a primeira página da lista de formulários, clique em Primeiro.
  • Para exibir a página anterior na lista de formulários, clique em Anterior.
  • Para exibir a página seguinte na lista de formulários, clique em Avançar.
  • Para exibir a última página da lista de formulários, clique em Último.
