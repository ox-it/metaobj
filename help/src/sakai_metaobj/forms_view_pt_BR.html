Visualizar um Formulário

Para ver um formulário conforme ele é exibido aos participantes do site, acesse a ferramenta de Repositório e siga estes passos:

  1. Se a página inicial do Repositório não for exibida, clique no botão Reset.
  2. Clique no link Adicionar de qualquer pasta em Repositório neste site.
  3. Selecione o item de Formulário do Adicionar Tipo de Item na lista.
  4. Usando a lista de opções para Selecionar o Tipo de Formulário, selecione o formulário.

    5. O formulário que é exibido aos participantes do site contém apenas duas das seções exibidas na tela. As duas seções que são exibidas aos participantes do site são Item do Formulário e Propriedades. A seção Adicionar Item (na parte superior da tela) e a seção de Acesso (na parte inferior da tela) não são exibidas para os participantes que utilizam o formulário.

  5. Para ver um outro formulário, repita o passo 4.
  6. Para sair, clique em Cancelar. O Sakai exibe a página inicial do Repositório.
