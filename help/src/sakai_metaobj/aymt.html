Requesting global publication of a form

If you have created a form that has general applicability, you may want to make it available for use in all sites. However, only system administrator(s) can publish a form globally (that is, across all sites), so you must send the administrator a request. To request that a form be published globally, in the Forms tool:

  1. If the Manage Forms page is not displayed, click the reset button ( 
     ).

  2. On the Manage Forms page, find the form you want to submit to the system administrator for global publication and click Publish Global.

  3. To proceed with the request, click Yes. The Manage Forms page will display. The Global State for this form is changed to "waiting for approval."
