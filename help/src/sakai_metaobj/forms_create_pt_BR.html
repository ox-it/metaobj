Criar um Formulário

Etapas Preliminares

Antes de criar um formulário, siga estes passos:

  1. Reveja formulários globais existentes em sua instância CLE. Você pode encontrar um formulário que se adapte aos seus propósitos, caso em que você não precisa criar um novo.
  2. Planeje o conteúdo cuidadosamente. Mantenha em mente os seguintes pontos:
    • Os nomes dos campos são exibidos à esquerda dos campos de texto a que se aplicam.
    • Nomes de campos longos são difíceis para os participantes ler.
  3. Considere a criação de um novo formulário em uma instância CLE teste para ver e alterá-lo conforme necessário antes de incluí-lo em sua instância de produção CLE.

Criar um Formulário

Para criar um formulário, acesse a ferramenta Formulários e siga estes passos:

  1. Se você não visualizar a barra de menu Formulários, clique no botão Reset.
  2. Clique em Adicionar na barra de menu. O Sakai exibirá a tela de Adicionar Formulário.
  3. Na caixa Nome, digite um nome significativo para o formulário.

    Escolha o nome cuidadosamente: Certifique-se de escolher um nome que indica o uso específico deste formulário. Depois que o formulário foi adicionado ao site ou publicado globalmente, seu nome será visível no Repositório para todos os participantes do site (ou se publicou globalmente, a todos os participantes de todos os sites nesta instância CLE). Portanto, se o formulário está adaptado a uma célula da matriz particular ou página do assistente, um nome genérico, como reflexão é enganoso e faz com que o formulário parece ser adequado para uso geral, quando, de fato, não é. Use um nome genérico para um formulário somente se os campos, nomes de campo, e as instruções para utilização de forma adequada em várias de situações.

  4. Clique em Selecionar Arquivo do Esquema. O Sakai exibirá a tela de Selecionar Item.
  5. Siga as instruções para selecionar um item. Depois que você clicar em Concluir, Sakai exibirá a tela de Adicionar Formulário novamente. O nome do arquivo selecionado é mostrado na caixa de arquivo de esquema (.xsd).
  6. Na caixa de Instrução, digite as instruções adequadas para o preenchimento do formulário. Use os ícones e menus suspensos para formatar o texto conforme o caso.
  7. Para adicionar o formulário para este site, clique em Adicionar Formulário. O Sakai cria o formulário e exibirá novamente a página inicial de Formulários. O novo formulário foi adicionado à lista no estado não publicado. Ou, para retornar à página inicial, sem acrescentar este formulário, clique em Cancelar.
