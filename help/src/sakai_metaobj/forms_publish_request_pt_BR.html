Solicitando que um formulário seja Publicado Globalmente

Se você criou um formulário que tem aplicabilidade geral, você convém torná-lo disponível para uso em todos os sites. No entanto, apenas o administrador do sistema Sakai pode publicar um formulário globalmente (isto é, em todos os sites do Sakai).

Para solicitar um formulário que seja publicado globalmente, acesse a ferramenta Formulários e siga estes passos:

  1. Se a página inicial de Formulários não for exibida, clique no botão Reset.
  2. Encontre o formulário que você deseja enviar para o administrador do sistema para publicação global.
  3. Clique no link Publicar para o formulário. O Sakai exibirá esta tela de confirmação:
  4. Para prosseguir com o pedido, clique em Sim, O Sakai exibirá a página inicial de Formulários. O Estado Global para este formulário foi alterado para "aguardando aprovação" Ou, para voltar à página inicialsem submeter o pedido, clique em Não.
