Alterar um Formulário

Usando esta ferramenta, você pode alterar o nome de um formulário e as instruções para utilizá-la. No entanto, você não pode usar essa ferramenta para alterar o arquivo esquema do próprio formulário.

Para alterar o nome ou instruções de um formulário, acesse a ferramenta Formulários e siga estes passos:

  1. Se você não consulte a página inicial de Formulários, clique no botão Reset.
  2. Encontre o formulário que deseja alterar, e clique no link Revisar, abaixo do seu nome. O Sakai exibirá a tela Revisar Formulário.
  3. Altere as entradas nas caixas Nome e Instrução, conforme necessário. Use os ícones e menus suspensos para formatar o texto instruções que desejar.
  4. Para prosseguir, clique em Salvar Alterações. O Sakai alterará o formulário e exibirá novamente a página inicial de Formulários. Ou, para retornar à página inicial sem alterar este formulário, clique em Cancelar.
