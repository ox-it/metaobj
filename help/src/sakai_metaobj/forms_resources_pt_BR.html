A Ferramenta Formulários e a Ferramenta Repositório

Se você é o organizador do site para um site de portfólio, a ferramenta de formulários permite a você criar e publicar formulários para uso neste site. Cada formulário que você criar para o site é baseado em um esquema xml(arquivo .xsd). Este arquivo é armazenado no Repositório.

Importante: Recomendamos que você armazene todos os arquivos .xsd no Repositório em Meu Site, em vez do portfólio. Participantes do site não podem trabalhar ou ver esses arquivos. Ao armazenar arquivos .xsd em sua área do Repositório pessoal (de preferência em uma pasta chamada Formulário .XSDs ou algo similar), você impede que os participantes do site, eventualmente, fiquem confusos. Esta prática deixa a área do Repositório do Site livre para ser utilizada como um lugar para arquivos que são para todos os participantes do site. Ele também permite que você use uma única cópia de um arquivo .xsd do seu Meu Site para criar formulários em todos os seus sites.

Todos os formulários também são armazenados em Repositório, para que os usuários possam acessá-los através da ferramenta Repositório. Em Repositório, o nome de cada formulário local é apresentado juntamente com o nome do site ao qual ele pertence, entre parênteses. Assim, um formulário intitulado "Educação" que é usado apenas no site dos Serviços de Carreira está listado em Repositório como a Educação (Serviços de Carreira).

Usando a ferramenta Formulários, você importa arquivos .xsd a partir de Repositório, dando nomes significativos, fornecendo instruções para usá-los, criando formulários, e publicando-os para uso dos participantes do site. Você também pode usar essa ferramenta para exportar formulários de um site e importá-los para outro.

A ferramenta Formulários não fornecer acesso direto aos formulários que você criar. Em vez disso, participantes do site acessam os formulários que eles precisam via ferramenta Repositório. Eles podem ver os formulários preenchidos em Repositório e nos assistentes, matrizes e portfólios que fazem uso desses formularios. Quando os usuários acessam a parte do assistente, matriz ou portfólio que contém um formulário, o formulário será exibido automaticamente.
