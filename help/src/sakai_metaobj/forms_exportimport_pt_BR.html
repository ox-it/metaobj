Exportar e Importar um Formulário

Se você possuir mais de um site, talvez você queira usar um formulário especial em dois ou mais sites. Para fazer isso, você pode recriar o formulário a partir do arquivo .xsd do Repositório ou exportar o formulário para a sua área de trabalho como um arquivo compactado e depois importá-lo para outro site (s) que você pretende utilizá-lo. Você também pode adquirir arquivos zip exportados para os formulários de outros usuários para importação para o seu site (s).

Exportar um Formulário

Para exportar um arquivo zipado para um formulário, acesse a ferramenta Formulários e siga estes passos:

  1. Se a página inicial de Formulários não for exibida, clique no botão Reset.
  2. Encontre o formulário que você deseja exportar.
  3. Clique no link Exportar abaixo do nome do formulário. O Sakai exibirá a janela Abrir Arquivo.
  4. Siga as instruções.

Importar um Formulário

Para importar um formulário, acesse o site ao qual você deseja importar e siga estes passos:

  1. Adicione o arquivo no Repositório.
  2. Acesse a ferramenta Formulários. Se a barra de menu de Formulários não for exibida, clique no botão Reset.
  3. Clique no botão Importar na barra de menu. O Sakai exibirá a tela de Formulário de Importação.
  4. Clique em Selecionar Arquivo. O Sakai exibirá a tela Selecionar Item.
  5. Siga as instruções dadas para selecionar um item. Depois que você clicar em Concluir, o Sakai exibirá a tela de Formulário de importação novamente. O nome do arquivo que você escolheu é exibido na caixa de Importar Arquivo.
    6. Clique no botão Importar. O Sakai importará o arquivo da área do Repositório e exibirá novamente a página inicial deFormulários. O formulário importado está listado. Ele tem o mesmo nome e instruções como tinha em seu site original. Ou, para retornar à página inicial, sem importar o arquivo para a ferramenta Formulários, clique em Cancelar. Independentemente do seu estado (publicado ou não publicado) no site original, o formulário que você importou está no estado não publicado sobre seu novo site.
